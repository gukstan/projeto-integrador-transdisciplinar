Esse projeto funciona como se fosse uma loja de cupcakes em web. É um projeto feito para o curso de Engenharia de Software da UNICSUL, na matéria Projeto Integrador Disciplinar II.
É importante ressaltar que não é um projeto real de alguma loja, e sim um trabalho acadêmico.
